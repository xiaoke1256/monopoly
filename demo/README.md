# 大富翁 Demo

## 运行方式

直接用浏览器打开 `index.html` 即可开始游戏（Chrome / Edge 均可）。

## 已实现功能（Demo）

- 开始页：玩家人数、AI 数量、初始资金、回合上限配置
- 棋盘与玩家移动（20 格循环地图）
- 回合流程：掷骰 -> 落点结算 -> 购买/升级 -> 结束回合
- 地产系统：购买、升级（最高 3 级）、过路费
- 特殊格：起点奖励、事件格、问答格、税收格、监狱
- 事件系统：资金、位移、状态、地产共 8 类示例事件
- 问答系统：历史/地理自然单选题、限时、奖励/惩罚、正确率统计
- 基础 AI：自动掷骰、买地、升级、结束回合、自动答题
- 破产判定与结算页（按总资产排名）
- 本地存档：对局中自动保存，首页可恢复

## 说明

这是一个“可玩闭环”的前端 demo，重点覆盖需求文档中的 MVP 核心玩法流程，便于后续继续扩展（更多事件、题库、动画和 UI 打磨等）。
